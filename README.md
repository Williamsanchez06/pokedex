# Consumo de API de Pokémon con React

## Descripción
Esta aplicación de React utiliza la API de Pokémon a través de GraphQL para mostrar una lista de todos los Pokémon y sus detalles. Los usuarios pueden buscar Pokémon por nombre o número, ver su información detallada, agregar Pokémon a una lista de favoritos y filtrar la lista por tipo.

## Tecnologías Utilizadas
- React
- GraphQL
- Apollo Client
- HTML5
- CSS3
